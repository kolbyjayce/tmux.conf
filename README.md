# tmux.conf

This is my personal tmux configuration file that does a couple things

1. Set prefix to ctrl-a
2. Bing pane navigation to vim navigation
3. Load tpm
4. Load catppuccin and set some formatting

## Using this config

- Clone this repo to `~/.config/tmux`

```bash
git clone --recurse-submodules https://github.com/kolbyjayce/tmux.conf.git ~/.config/tmux
```

- install plugins with `ctrl-a + I`
- reload config with `ctrl-a + r`
